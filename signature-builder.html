Setup

Install it in two minutes.

Gmail

  1. Click Copy signature.
  2. Settings → See all settings → General → Signature → Create new.
  3. Paste. Set as default for new emails and replies.
  4. Scroll down, Save changes.

Apple Mail

  1. Click Copy signature.
  2. Mail → Settings → Signatures → + for your Sagepilot account.
  3. Paste, and untick “Always match my default message font”.

Outlook

  1. Settings → Mail → Compose and reply.
  2. Paste into the signature editor and save.
